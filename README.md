
# phrasecloudjs ![npm version](https://badge.fury.io/js/phrasecloudjs.svg)
HTML JavaScript generated based on [wordcloud2.js](https://github.com/timdream/wordcloud2.js)

## Installation

* script：

```html
<script src="dist/wordCloudJS.min.js"></script>
```
* npm：

```bash
npm i phrasecloudjs
```
